1.	What is the difference between var, let and const?  
Answer:  
i) Var is  function scoped, it can be re-declared and updated.  
ii) Let is blocked-scoped, can be updated but not re-declared in the same scope.  
iii) Const also blocked-scoped cannot be updated or re-declared, the value of const is constant.  

3.	What is the spread operator(…)?  
Answer: Spread operator spread or expand arrays or object into individual elements.  

4.	What is the different between map(), filter() and forEach()?  
Answer:  
i) map() function loop through and return a new array after transforming each element.  
ii) filter() function loop through and return a new array with the elements that pass the condition.   
iii) forEach() function is a loop, its loops through elements, doesn’t return a new array.
